# WSL Dotfiles

1. Install All Packages

```
./install.sh
```

2. Setup Bash

```
./setup.sh
```

3. Set WSL-Terminal Theme

```
Right click wsl-terminal -> Click Options -> Looks -> Select Theme 'afterglow.minttyrc'
```

4. Setup fonts

```
sudo apt-get install fonts-powerline
```
